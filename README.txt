При запуске приложения откроется форма авторизации. Можно зайти под обычным пользователем, а можно в качестве администратора (для этого нужно ввести пароль "Admin").
В режиме обычного пользователя функционал будет ограничен - можно будет просматривать таблицу данных и выполнять запросы.
В режиме администратора можно будет также добавлять, изменять и удалять записи таблицы.
Сверху в меню есть пункт "Игра". Перейдя в него, вам будет предложено сыграть в игру. Правила игры изложены в пункте меню "Справка" окна "Game".
